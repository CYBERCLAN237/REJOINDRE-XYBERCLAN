 # Join XYBERCLAN Community

If you'd like to join the XYBERCLAN Community, please submit a request by creating an issue using the template below.

## How to Request to Join

1. Go to the **Issues** tab.
2. Click on **"New Issue."**
3. Select the **Join Request** template.
4. Fill in your full name, email address, and a brief explanation of why you want to join.
5. Click **"Submit new issue."**

We will review your request and send you an invitation!
